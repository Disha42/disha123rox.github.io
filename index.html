Sign Language Recognition

The software aims at real time detection of various gestures performed by the user, processing them and producing the output in the form of speech. The project has been done in two phases.
Phase 1 includes the processing of the 2-D gestures and converting to speech.We built a linux based application using C++, OpenCV libraries.
Phase 2 includes the processing of the 3-D signs used in the sign language and converting to speech.
We used Microsoft Kinect XBOX 360 to detect the the gestures and intercepted the gestures using C# code, 20 joint skeletal tracking system for Kinect in the Visual Studio. The software aims to help the deaf and dumb people of the society to communicate easily.
